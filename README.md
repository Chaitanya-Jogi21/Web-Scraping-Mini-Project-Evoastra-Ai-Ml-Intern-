# Web Scraping Mini Project – Evoastra AI/ML Intern

## Project Overview
This project demonstrates web scraping techniques to extract structured data from websites and generate automated reports and presentations. The main goal is to collect meaningful data, organize it, and visualize it in a professional format. This project is implemented using **Python** in **Jupyter Notebook**.

## Features
- Scrapes data from websites (e.g., Honda cars).
- Saves extracted data in structured formats like CSV, Excel, or JSON.
- Automatically generates a detailed report (`.docx`) and a presentation (`.pptx`).
- Easily extendable to other websites and datasets.

## Requirements
- Python 3.x
- Jupyter Notebook
- Libraries:
  - `requests`
  - `beautifulsoup4`
  - `pandas`
  - `openpyxl`
  - `python-pptx`
